### About This Project
Got too many Enders laying around? Don't need a [5th Trident?](https://github.com/yell3D/Ender3dent) Well! Try the Ender3CNC, born from a lack of sleep, a wish to learn more CAD and too many spare parts floating around.

</div>

### Reused from the Ender 3
* Frame
* steppers
* wires (might need to extend some)
* leadscrew
* leadscrew nut
* coupler
* mcu
* psu
* most of the nuts and bolts
* v-wheels
* alu spacers
* belts (see BOM)
* limit switches (see BOM)
* screen if you want

### Features
* BOM cost approximately $60 (not counting the Ender 3 PRO or dremel/spindle)
* Chances are you have some of the BOM laying around already.
