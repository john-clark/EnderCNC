# Tools

| Tool | Notes |
|------|-------|
| M5 Tap | For cutting threads in aluminum extrusions |
| Hex Drives | Standard metric hex keys |
| Dremel or spindle | Check compatibility with clamps |
| Printed press-fit pulley extraction tool | Optional: can cut pulleys if unavailable |
| Saw | To cut the aluminum extrusion |

**💡 Tip:** Use a flat surface for squaring the frame.
